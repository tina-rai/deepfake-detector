# DeepScan

DeepScan is a web-based deepfake detection system built using Flask and TensorFlow.

## Features

- Image Deepfake Detection
- Video Deepfake Detection
- Face Detection
- AI Attention Heatmap
- Edge Artifact Visualization
- Image Quality Analysis
- Consistency Analysis
- Risk Level Estimation
- Confidence Score
- Forensic Report
- Responsive Web Interface

## Tech Stack

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- NumPy
- HTML
- CSS

## How it Works

1. User uploads an image or video.
2. Faces are detected using OpenCV.
3. The trained CNN predicts whether the face is Real or Fake.
4. Forensic analyses are generated.
5. A detailed AI report is displayed.
